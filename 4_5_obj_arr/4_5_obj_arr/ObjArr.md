## 객체 배열

- 객체 기반의 배열은 당므의 형태로 선언 (SoSimple 이 클래스 이름)

```cpp
SoSimple arr[10];
```

- 동적 할당하는 경우

```cpp
SoSimple * ptrArr = new SoSimple[10];
```

이런 식으로 선언하게 되면 10개의 SoSimple 객체가 모여서 배열을 구성한다.
하지만 이런 식으로 선언하게 되는 경우에도 생성자는 호출 되며, 이 때 선언과정에서 생성자는 별도로 명시되지 못한다.
따라서 위의 형태로 배열이 생성 되려면 다음 형태의 생성자가 반드시 정의 되어있어야한다.

```cpp
SoSimple { ... }
```

배열 선언 이후에 각각의 요소를 원하는 값으로 초기화하길 원한다면 일일이 초기화의 과정을 별도로 거쳐야한다.
